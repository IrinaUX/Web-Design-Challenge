Latitude

Weather temperatures visualizations based on the weather data of 500+ cities across the world of varying distance from the equator. OpenWeatherMap API used to create a representative model of weather across world cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. Latitude. Factors we looked at included: temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlation witnessed.
